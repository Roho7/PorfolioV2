PROJECT

Lo-Fi Station

WEB-DEV

  HTML   CSS   Javascript

2023
CONTEXT

I always found it to be distracting to search for good lo-fi music before studying or relaxing. It sent me down a spiral not being able to find the perfect one for my mood. While studying, my pomodoro timer would be on an app and the music would be on my computer and I would be completely thrown off when one of them started or stopped playing. So I integrated these two features into this website and made it into an immersive lo-fi experience that automatically generates music according to the selected mood.
